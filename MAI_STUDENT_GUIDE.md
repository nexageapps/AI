## Study Strategies for MAI Students

### 1. Before Semester Starts
**Goal:** Build strong foundations

- Complete B01-B07 (Foundation & Core ML)
- Get comfortable with TensorFlow/PyTorch
- Set up your development environment
- Review linear algebra and calculus concepts

**Time Investment:** 2-3 weeks, 2-3 hours per day

### 2. During Semester One
**Goal:** Complement coursework with practice

**Week 1-4: Machine Learning Fundamentals**
- Review B02-B04 alongside COMPSCI 762 lectures
- Complete assignments in B14 related to classification
- Join study groups and discuss implementations

**Week 5-8: Neural Networks Deep Dive**
- Work through B05 and B09 as you learn about deep learning
- Implement variations of architectures discussed in class
- Compare your implementations with course materials

**Week 9-12: Advanced Topics**
- Explore B10-B11 for RNNs and Transformers
- Start thinking about your research project
- Use B15 for capstone project ideas

### 3. During Semester Two
**Goal:** Specialize and build portfolio

- Choose a specialization (NLP, Computer Vision, etc.)
- Complete relevant capstone projects from B15
- Contribute to open-source AI projects
- Prepare for your dissertation/thesis

### 4. Exam Preparation
**Goal:** Consolidate knowledge

- Review all notebooks sequentially
- Implement algorithms from scratch (no libraries)
- Explain concepts to study partners
- Complete practice assignments from B14

## How to Use This Repository Effectively

### For Lecture Preparation
**Before Class:**
1. Read the relevant notebook (e.g., B09 before CNN lecture)
2. Run the code and observe outputs
3. Note down questions for the lecture
4. Try modifying parameters to see effects

**After Class:**
1. Revisit the notebook with new understanding
2. Complete the related assignment from B14
3. Implement variations discussed in lecture
4. Share your implementations with classmates

### For Assignments
**When You Get an Assignment:**
1. Review related notebooks for implementation patterns
2. Use the code as reference, not copy-paste
3. Extend the concepts to your specific problem
4. Document your approach and reasoning

**Pro Tip:** Professors appreciate when you go beyond basic requirements. Use B15 capstone projects as inspiration for assignment extensions.

### For Research Projects
**Choosing a Topic:**
1. Explore B15 capstone projects for ideas
2. Identify gaps in current implementations
3. Combine multiple concepts (e.g., CNNs + Attention)
4. Discuss with your supervisor

**Implementation Phase:**
1. Start with simplified versions from this repo
2. Gradually add complexity
3. Document everything (future you will thank you)
4. Version control with Git

## Common Challenges and Solutions

### Challenge 1: "The math is overwhelming"
**Solution:**
- Focus on intuition first (this repo emphasizes understanding)
- Implement before diving deep into proofs
- Use visualizations (all notebooks include plots)
- Review linear algebra/calculus as needed

### Challenge 2: "I don't know where to start"
**Solution:**
- Follow the sequential order: B01 → B02 → ... → B15
- Don't skip fundamentals (B01-B04)
- Each lesson builds on previous ones
- Ask for help in study groups

### Challenge 3: "My code doesn't work"
**Solution:**
- Compare with notebook implementations
- Use print statements to debug
- Check tensor shapes (most common issue)
- Ask on course forums or Discord

### Challenge 4: "I'm falling behind"
**Solution:**
- Prioritize coursework over this repo
- Use notebooks for quick reviews
- Focus on understanding, not completion
- Reach out to course coordinators

## Frequently Asked Questions

### Q: Should I complete this repo before starting MAI?
**A:** Not necessary, but B01-B07 will give you a strong foundation. Focus on B01-B04 if you're new to ML.

### Q: Can I use code from this repo in my assignments?
**A:** Use it as reference and learning material, but write your own implementations. Understand the concepts, don't copy-paste.

### Q: How much time should I spend on this repo?
**A:** 3-5 hours per week alongside your coursework. Prioritize your course assignments first.

### Q: What if the repo content differs from lectures?
**A:** Always prioritize your course materials. This repo is supplementary. Discuss differences with your professors.

### Q: Can I share my course materials here?
**A:** No, respect copyright. Only share your own implementations and projects.
