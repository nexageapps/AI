# Symbolic Logic Explorer

Interactive propositional and first-order logic app for COMPSCI 713.

**🌐 Live:** [https://nexageapps.github.io/AI/symbolic-logic/](https://nexageapps.github.io/AI/symbolic-logic/)

## Features
- **Explore** — 8 real-world scenarios with emojis, stories, and challenges (Rainbow Rule, Pizza Party, De Morgan's, etc.)
- **Learn** — 10 step-by-step lessons covering propositional logic through FOL with kinship
- **Quiz** — 7 interactive questions with explanations across difficulty levels
- **First-Order Logic** — Interactive kinship world builder, free/bound variable identifier, FOL equivalence explorer
- **Playground** — Build custom expressions with ∧, ∨, →, ↔, ¬ operators
