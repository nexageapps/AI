# RNN Explorer — COMPSCI 713 Week 4

Interactive web application for learning Recurrent Neural Network concepts.

## Features

- **RNN Flow** — Animated visualization of data flowing through RNN/LSTM/GRU cells over time
- **Learn** — Step-by-step lessons covering RNN architecture, LSTM gates, GRU, bidirectional RNNs, and applications
- **LSTM Gates** — Interactive sliders to manipulate forget/input/output gates and see how they affect cell state and hidden state in real-time
- **Playground** — Text generation simulator, time series prediction, and sentiment analysis demos
- **Quiz** — 10-question quiz covering all RNN concepts

## Run Locally

```bash
npm install
npm start
```

## Deploy

Deployed automatically via GitHub Actions to GitHub Pages on push to `main`.
