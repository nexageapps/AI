## Quick Reference

### Key Formulas

**Neuron Output:**
```
y = g(wᵀx + b)
```

**Gradient Descent:**
```
w ← w - α∇L(w)
```

**Backpropagation (Chain Rule):**
```
∂L/∂wₗ = ∂L/∂aₗ × ∂aₗ/∂zₗ × ∂zₗ/∂wₗ
```

**Learning Rate Trade-off:**
```
α too small → slow convergence
α too large → divergence
α optimal → fast & stable
```

### Common Batch Sizes

- 32: Small networks
- 64: Good default
- 128: Balanced
- 256: Large networks
- 512: Very large networks

### Activation Functions

- **ReLU**: Hidden layers (most common)
- **Sigmoid**: Binary classification output
- **Softmax**: Multi-class classification output
- **Tanh**: Alternative for hidden layers
